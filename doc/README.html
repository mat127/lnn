2. Introduction

This document describes how to build and install the Lotus Notes Notifier. It is a simple Java agent intended for notification about incoming emails or any other database/document changes within the Lotus Domino server.

3. Requirements

3.1 Basic

  • Lotus Notes 6.5.x client

3.2 Optional - Jabber Notifier

  • Smack library

3.2 Optional - Building

  • Java 2 SDK
  • Ant

Building

Installation

  1. Download binary distribution of the Lotus Notes Notifier from http://sf.net/projects/lnn/ or build your own following steps described in section 4
  2. Start Lotus Notes, login and open the e-mail database
  3. Select View - Agents from menu, an list of agents working woth the e-mail database appears
